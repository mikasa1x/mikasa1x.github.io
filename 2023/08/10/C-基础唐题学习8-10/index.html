C++处理异常机制

检查、抛出和捕获（try，throw，catch）

该函数参数为1时返回值为（）

int GetResult(int a){
  int b = 0;
      __try{
          if ( a != 0 ){
             b++;
          }
          return b;
      }
    __finally{
        --b;
        }
    return b;
}

在函数体_ try内，return语句将变量b的值暂存，然后跳转到_ finally块，_ finally块中若无return，则再返回到_ try块中返回暂存值；

模板类

  • 可用来创建动态增长和减小的数据结构
  • 它是类型无关的，因此具有很高的可复用性。
  • 它在编译时而不是运行时检查数据类型，保证了类型安全
  • 它是平台无关的，可移植性
  • 可用于基本数据类型
